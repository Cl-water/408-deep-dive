# 408 Deep Dive

本仓库用于记录我在 408 计算机专业基础课程学习过程中的深度思考与技术文档沉淀。

与普通知识点整理不同，本仓库更关注：

- 概念为什么产生；
- 知识点之间是否存在逻辑关系；
- 教材中的定义如何从工程问题中抽象出来；
- 同一问题在不同视角下如何被描述；
- 如何用 Markdown、Mermaid、LaTeX 等工具进行结构化表达。
